## V2EX-android


### 支持功能

 * 话题查看
 * 节点查看和按字母搜索
 * 用户资料查看
 * 话题回复
 * 话题创建
 * 未读提醒查看
 * 常用分类节点话题浏览

### 其他功能

 * 实现了节点、话题、用户三个Scheme,通过话题、节点、用户链接直接打开客户端进行相关信息浏览
 * 对用户已经浏览过的话题作记忆处理并在界面上进行展示
 * 话题、帖子等采用了卡片式布局

### 编译需求

* JDK 1.7+
* Android Studio 1.0+
* Android SDK r21+
